## 🐼 中文 介绍

**嘿！是不是厌倦了那些花里胡哨的新标签页？** 🤯

想要 Chrome 原生的清爽界面，又离不开百度、必应或者 DuckDuckGo？这款扩展就是为你量身打造的！它保持了原生的极简风格，但功能更强大。而且，你还可以换上自己喜欢的壁纸，瞬间变身个性化桌面！🚀

### ✨ 核心亮点

*   🎨 **原生体验**：完美复刻 Chrome 原生新标签页布局，毫无违和感。
*   🔍 **一键切换**：搜索框右侧内置切换器，Google、百度、必应 (Microsoft Bing)、DuckDuckGo 随意切。
*   🌙 **自动深色模式**：晚上冲浪不刺眼，自动跟随系统变黑，Logo 也会智能变色哦。
*   🖼️ **自定义壁纸**：上传你最爱的图片作为背景！设置后，Logo 会自动隐藏，变身为一个超酷的极简时钟。
*   ⚡ **极速响应**：所有数据（包括壁纸）都存在本地，秒开无延迟！

### 🛠️ 如何安装

1.  下载或 Clone 本项目到本地。
2.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`。
3.  打开右上角的 **"开发者模式" (Developer mode)** 开关。
4.  点击左上角的 **"加载已解压的扩展程序" (Load unpacked)**。
5.  选择本项目所在的文件夹。
6.  搞定！打开新标签页试试吧！🎉

## 🦁 English Introduction

**Hey there! Tired of cluttered new tab pages?** 🤯

Want that clean, native Chrome look but love using Bing, Baidu, or DuckDuckGo? This extension is your new best friend! It keeps things simple, native, and super powerful. Plus, you can customize it with your own wallpapers to make it truly yours! 🚀

### ✨ Key Features

*   🎨 **Native Look**: Feels exactly like the original Chrome new tab, but better.
*   🔍 **Search Switcher**: Switch between Google, Baidu, Bing (Microsoft Bing), and DuckDuckGo with a single click inside the search bar.
*   🌙 **Dark Mode**: Automatically adapts to your system theme. Even the logos adjust to look great in the dark!
*   🖼️ **Custom Backgrounds**: Upload your favorite wallpaper! When set, the search logo vanishes and is replaced by a stylish, large clock.
*   ⚡ **Lightning Fast**: Everything, including your wallpaper, is stored locally. No lag, instant load!

### 🛠️ How to Install

1.  Download or Clone this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Toggle on **"Developer mode"** in the top right corner.
4.  Click **"Load unpacked"** in the top left.
5.  Select the folder containing this extension.
6.  Done! Open a new tab and enjoy! 🎉

## 🦊 日本語 紹介

**やあ！ごちゃごちゃした新しいタブページにうんざりしていませんか？** 🤯

Chromeのシンプルでネイティブな見た目は好きだけど、BingやDuckDuckGoも使いたい？そんなあなたにぴったりの拡張機能です！シンプルさを保ちつつ、機能はパワフルに。さらに、お気に入りの壁紙でカスタマイズして、あなただけのデスクトップに変身させましょう！🚀

### ✨ 主な機能

*   🎨 **ネイティブなデザイン**: Chrome純正の新しいタブページのような使い心地。違和感ゼロです。
*   🔍 **検索エンジン切替**: 検索バーの中から、Google、Baidu、Bing (Microsoft Bing)、DuckDuckGoをワンクリックで切り替え可能。
*   🌙 **ダークモード対応**: システムのテーマに合わせて自動的に色が変化。夜のブラウジングも快適です。
*   🖼️ **カスタム壁紙**: お気に入りの画像をアップロードして背景に設定！設定すると、ロゴが隠れてスタイリッシュな時計が表示されます。
*   ⚡ **超高速**: 壁紙を含むすべてのデータはローカルに保存されるため、読み込みは一瞬です！

### 🛠️ インストール方法

1.  このリポジトリをダウンロードまたはクローンします。
2.  Chromeを開き、アドレスバーに `chrome://extensions/` と入力します。
3.  右上の **「デベロッパーモード」 (Developer mode)** をオンにします。
4.  左上の **「パッケージ化されていない拡張機能を読み込む」 (Load unpacked)** をクリックします。
5.  この拡張機能のフォルダを選択します。
6.  完了！新しいタブを開いて試してみてください！🎉
